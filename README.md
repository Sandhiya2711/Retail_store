# 🛒 Retail Store Data Analysis

## 📌 Project Overview
This project performs data analysis and visualization on a retail store dataset using Python.  
It helps to understand sales performance, product trends, and relationships between key variables.

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset
The dataset file: `retail_store.csv`

### Columns:
- Product_ID
- Quantity
- Unit_Price
- Date

---

## 🔍 Features & Analysis

### ✅ Revenue Calculation
Revenue is calculated using:
```
Revenue = Quantity × Unit Price
```

### ✅ Monthly Analysis
- Converts Date column into datetime format  
- Extracts month from date  
- Calculates total monthly revenue  

### ✅ Product-wise Sales
- Groups data by Product_ID  
- Computes total revenue for each product  

### ✅ Correlation Analysis
- Finds relationship between:
  - Quantity
  - Unit Price
  - Revenue  

---

## 📊 Visualizations

The project generates the following charts:

### 📌 1. Bar Chart – Product Sales
Shows revenue generated by each product.

### 📌 2. Line Chart – Monthly Trend
Displays how revenue changes over months.

### 📌 3. Pie Chart – Product Share
Shows percentage contribution of each product.

### 📌 4. Heatmap – Correlation Matrix
Visualizes relationships between Quantity, Unit Price, and Revenue.
